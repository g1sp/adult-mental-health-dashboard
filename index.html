📚 How to Build This App

Learn web development by understanding how this Adult Mental Health Dashboard was built.

🎯 About This Application

This is an adult-specific adaptation of the mental health dashboard pattern, using real SAMHSA (Substance Abuse and Mental Health Services Administration) data for adults aged 18 and above.

Key Features:

  • Interactive data visualization with Chart.js
  • Evidence-based treatment browser
  • Adult risk factors and causes
  • Responsive design for all devices
  • Real health statistics from trusted sources

📊 Data Sources

  • SAMHSA NSDUH - National Survey on Drug Use and Health (annual adult statistics)
  • NIMH - National Institute of Mental Health research
  • CDC - Centers for Disease Control health data
  • FDA - Medication and treatment approvals
  • APA - American Psychological Association clinical guidelines

🏗️ Architecture

This dashboard follows a clean separation of concerns:

  • data-handler.js - Loads and filters statistics
  • chart-manager.js - Creates visualizations
  • treatment-handler.js & treatment-ui.js - Treatment browser
  • risk-factors-handler.js & risk-factors-ui.js - Risk factors display
  • app.js - Main orchestrator

💻 Key Technologies

  • HTML5 - Semantic markup
  • CSS3 - Responsive design with Bootstrap 5
  • Vanilla JavaScript - No frameworks needed
  • Chart.js - Beautiful data visualizations
  • JSON - Data storage format

🔑 JavaScript Concepts Used

  • ES6 Classes - Organized, reusable code structure
  • Async/Await - Load data without blocking
  • Event Listeners - React to user interactions
  • DOM Manipulation - Update page dynamically
  • Array Methods - Filter, map, reduce data
  • Promises - Handle async operations

📈 Working with Data

The dashboard processes data through multiple stages:

  1. Load JSON data from files
  2. Apply user filters (year, age, condition)
  3. Aggregate data for visualizations
  4. Render charts with Chart.js
  5. Update tables and cards dynamically

🎨 Responsive Design

Bootstrap 5 and CSS Grid ensure the app works on any device:

  • Mobile: Single column, touch-friendly
  • Tablet: Two columns, optimized spacing
  • Desktop: Full multi-column layouts
  • Charts resize automatically
  • Touch-friendly controls on all sizes

🚀 Deployment

This app requires no build tools or backend server:

  • No npm installation needed
  • No database required
  • No API keys needed
  • Deploy anywhere: GitHub Pages, Netlify, or traditional hosting
  • Open index.html in any modern browser
🏆 Learning Objectives

By studying this codebase, you'll learn:

  • How to build real-world data applications
  • Clean code architecture patterns
  • Working with third-party libraries
  • Responsive web design principles
  • Best practices for performance
  • How to handle real data responsibly
